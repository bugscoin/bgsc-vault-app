# BGSC Vault App

Web3 staking vault interface for BGSC token on BNB Chain.

## Features

- Connect wallet (RainbowKit)
- Deposit/withdraw BGSC tokens
- View staking rewards
- Multi-language support (EN/KO)

## Tech Stack

- React
- Wagmi/Viem
- RainbowKit
- Ethers.js
- Styled Components

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
